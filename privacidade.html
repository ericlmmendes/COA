Política de Privacidade

Introdução

A ERICLM, responsável pelo Controle Operacional por Agência (COA), valoriza a privacidade dos seus usuários. Esta Política de Privacidade explica como coletamos, usamos e protegemos as informações fornecidas por você ao usar nossa aplicação.

Dados Coletados

O COA coleta apenas os dados inseridos voluntariamente pelo usuário, como informações sobre fornecedores, turnos, quantidades de colaboradores, faltas, atestados e nomes de colaboradores. Esses dados são processados localmente no seu dispositivo e enviados via WhatsApp para compartilhamento, conforme solicitado pelo usuário.

Uso dos Dados

Os dados inseridos são utilizados exclusivamente para gerar relatórios operacionais e enviá-los via WhatsApp, conforme a funcionalidade da aplicação. Não armazenamos, processamos ou compartilhamos esses dados em servidores externos.

Segurança

Como o COA opera no lado do cliente (navegador), todos os dados são processados localmente. Recomendamos que os usuários evitem inserir informações sensíveis que não desejem compartilhar via WhatsApp, pois o envio depende de uma plataforma externa.

Alterações na Política

Podemos atualizar esta Política de Privacidade periodicamente. Qualquer alteração será publicada nesta página, e a data de atualização será revisada.
